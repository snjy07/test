
## About Project

Laravel based simple machince testing task related to product and order management project.

## Migration to local environment
- First copy that project and put it to your local environment.
- Change database credentials in .env file.
- Open project in command line and run command 'php artisan migrate'.
- Serve that project to your local environment using command 'php artisan serve'.
- To test open link 'https://you-local-project-url/administrator'.
